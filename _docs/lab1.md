プロンプト1
```
マイクロソフトの沼本 健（ぬもと たけし）さんについて、
マイクロソフトにおける役割と経歴、
最近の取り組みや発信（登壇・記事・SNS など）を教えてください。
全部で300文字程度で簡潔にまとめてください。
```
